How pixel ( px )and percentage ( % ) unit works, When an element is positioned absolutely.

When an element is positioned absolutely, its position is determined relative to its closest positioned ancestor (or the initial containing block ( body ) if no positioned ancestor is found ).

Pixel unit ( px )

When you use pixel (px) units for positioning an absolutely positioned element, the element will be positioned a fixed number of pixels from the top, bottom, left, or right of its containing block. For example, top: 20px; will position the element 20 pixels from the top of its containing block.

Percentage unit ( % )

When you use percentage (%) units for positioning an absolutely positioned element, the element will be positioned a percentage of the height or width of its containing block. For example, top: 50%; will position the element vertically at 50% of the height of its containing block.

What if no top, bottom, right, or left properties are assigned to an absolutely positioned element.

7

If no top, bottom, right, or left properties are assigned to an absolutely positioned element, the element will be positioned at the default location based on the normal document flow. This means that the element will be positioned where it would normally appear in the document, but it will be taken out of the normal document flow and positioned absolutely relative to its closest positioned ancestor.
