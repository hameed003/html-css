CSS Pseudo Elements

A CSS pseudo-element is used to style specified parts of an element.

For example, it can be used to:

  • Style the first letter, or line, of an element
  • Insert content before, or after, the content of an element

::first-letter

The ::first-letter pseudo-element is used to add a special style to the first letter of a text.

IMPORTANT NOTE:

The ::before and ::after pseudo-element, by default, is an inline element and does not have any content or specific display properties that would make it take up space or be visible on the screen.

To make the ::before and ::after pseudo-element visible, you typically need to set its content property to a non-empty value (such as text or an image) or use other properties like display: block; , background-color , border or box-shadow to give it a visual representation on the page.

Use the background-image property to add an image to the ::before and ::after element. This will make the element visible with the image as its background.

::selection

The ::selection pseudo-element matches the portion of an element that is selected by a user.

The following CSS properties can be applied to ::selection: color, background, cursor, and outline.
